# Backend Service для Landing Page Разработчика

Это тестовое задание реализовано на фреймворке **FastAPI (Python)**. 

Оно представляет собой полноценный REST API сервис с архитектурным разделением слоев, валидацией, логированием, механизмом защиты от спама (Rate Limiting) и интеграцией с OpenAI.

## Особенности реализации
- **Архитектура:** Использован слоистый паттерн проектирования (`endpoints` -> `services` -> `repositories`). Это позволяет легко менять хранилище (например, перейти с файловой базы данных на PostgreSQL) без изменения бизнес-логики.
- **Интеграция с AI (OpenAI):** AI анализирует тональность сообщения (sentiment) и генерирует автоответ для пользователя.
- **Отказоустойчивость (Graceful Fallback):** Если API-ключ OpenAI не указан или API OpenAI недоступно, сервис не падает. Он возвращает дефолтный ответ, чтобы форма пользователя успешно отправилась.
- **Защита (Rate Limiter):** Реализован кастомный in-memory / file-based лимитер запросов по IP пользователя, защищающий от DDoS/спам-атак. Ограничения настраиваются через переменные окружения.
- **Обработка ошибок:** Добавлен глобальный Exception Handler, возвращающий красивые JSON ответы без утечки Stacktrace. Запросы логируются в `data/app.log`.
- **Встроенный Frontend (Бонус):** В проект добавлен визуально красивый лендинг (Dark Mode, Glassmorphism, асинхронные запросы через Fetch API). Он раздается напрямую из FastAPI через `StaticFiles`, что позволяет проверить работу проекта прямо в браузере без сторонних утилит.

## Стек технологий
- **Backend:** Python 3.9+, FastAPI, Uvicorn
- **Frontend:** HTML5, CSS3, Vanilla JS
- **Валидация:** Pydantic (email-validator)
- **AI Инструменты:** `openai` (Official Python SDK)
- **Хранение данных:** Файловая система (папка `data/`, JSON-файлы для статистики)

## Как запустить проект

1. **Клонирование и переход в папку**
   ```bash
   # Предполагается, что вы уже находитесь в папке проекта
   ```

2. **Создание виртуального окружения и установка зависимостей**
   ```bash
   python -m venv venv
   # Активация на Windows
   venv\Scripts\activate
   # Активация на Linux/Mac
   source venv/bin/activate
   
   pip install -r requirements.txt
   ```

3. **Настройка переменных окружения**
   Скопируйте пример файла конфигурации:
   ```bash
   cp .env.example .env
   ```
   Откройте `.env` и впишите свой `OPENAI_API_KEY`, если хотите использовать реальную нейросеть. В противном случае оставьте `mocked_key` (или пустое значение), и сработает *fallback-режим*.

4. **Запуск сервера**
   ```bash
   uvicorn app.main:app --reload
   ```

5. **Документация API (Swagger UI)**
   Откройте в браузере: `http://127.0.0.1:8000/docs`

## Описание API (Эндпоинты)

### 1. Отправка формы обратной связи
**POST** `/api/contact`
```json
// Запрос (Body)
{
  "name": "Иван Иванов",
  "phone": "+79991234567",
  "email": "ivan@example.com",
  "comment": "Привет! Отличное портфолио, давайте работать вместе."
}

// Ответ (201 Created)
{
  "status": "success",
  "message": "Ваша заявка успешно отправлена.",
  "ai_analysis": {
    "sentiment": "positive",
    "auto_reply": "Здравствуйте, Иван Иванов! Спасибо за теплые слова..."
  }
}
```

### 2. Проверка статуса
**GET** `/api/health`
```json
{
  "status": "ok"
}
```

### 3. Получение статистики
**GET** `/api/metrics`
```json
{
  "total_requests": 1,
  "requests": [
    {
      "name": "Иван Иванов",
      "email": "ivan@example.com",
      "sentiment": "positive",
      "timestamp": "2026-07-22T10:30:00.000Z"
    }
  ]
}
```
